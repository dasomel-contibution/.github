# Welcome to dasomel-contibution

A collection of **open-source contribution activities**, focusing on the **eGovFrame (전자정부 프레임워크)** ecosystem and public sector software.

## About

This organization tracks and archives open-source contributions made to Korean public sector frameworks — primarily eGovFrame — including bug fixes, sample applications, and template improvements.
